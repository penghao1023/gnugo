SGF 文件格式

这是由飞燕围棋安卓版制作的 SGF FF[4] 规格说明书。如有出入，以英文版原文为准，原文地址: http://www.red-bean.com/sgf

SGF 是'Smart Game Format'的缩写。此文件格式是为了存储双人棋类对局记录而设计的。它是纯文本格式，基于树形结构，所以对局存盘文件可以方便地使用电子邮件发送及使用基于文本的传输协议传送。
使用SGF文件格式的目的是为了存储对局并且易于提供注释和分析。(例如：标记，分支等)

最近更新: 2003年9月26日。(原文Last updated: 2001-08-12 )

注意: 请区别不同的命令词（必须、必须不）和推荐词（建议、应当、不应当）。命令词“必须”和“必须不”是必须遵守的要求，如果应用不遵守命令词的规定，就会导致错误。推荐词“建议”、“应当”、和“不应当”不是必须遵守的规定，但应用软件如不遵守推荐词的规定，就会导致不良的风格。
